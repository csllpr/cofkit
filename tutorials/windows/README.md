# Native Windows Jupyter tutorials / 原生 Windows Jupyter 教程

- [English Windows tutorials](en/README.md)
- [Windows 中文教程](cn/README.md)

These notebooks use regular Python cells; neither PowerShell nor WSL is required. Conda and cofkit commands are passed as argument lists through Python's `subprocess` module, avoiding shell parsing. Start Jupyter from a Miniforge, Miniconda, or Anaconda installation so that `conda` is available.

这些 Notebook 使用普通 Python 单元格，不需要 PowerShell 或 WSL。Conda 和 cofkit 命令通过 Python 的 `subprocess` 模块以参数列表传递，从而避免 Shell 解析。请从 Miniforge、Miniconda 或 Anaconda 环境启动 Jupyter，确保可以使用 `conda`。
