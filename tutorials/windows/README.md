# Native Windows Jupyter tutorials / 原生 Windows Jupyter 教程

- [English Windows tutorials](en/README.md)
- [Windows 中文教程](cn/README.md)

These notebooks use Windows PowerShell through `%%script powershell -NoProfile`; WSL is not required. Start Jupyter from a Miniforge, Miniconda, or Anaconda PowerShell prompt so that `conda` is available.

这些 Notebook 通过 `%%script powershell -NoProfile` 使用 Windows PowerShell，不需要 WSL。请从 Miniforge、Miniconda 或 Anaconda PowerShell 提示符启动 Jupyter，确保可以使用 `conda`。
