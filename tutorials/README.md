Every edition is CLI-first and includes commented Python API equivalents. The native-Windows notebooks use regular Python cells with `subprocess` argument lists and require neither PowerShell nor WSL.

所有版本均以 CLI 为主，并包含注释掉的 Python API 等价实现。原生 Windows Notebook 使用普通 Python 单元格和 `subprocess` 参数列表，不需要 PowerShell 或 WSL。
