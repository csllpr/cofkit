# Jupyter tutorials

Language: **English** | [中文](../cn/README.md)

Run Jupyter from the repository root and follow the notebooks in order:

1. [`01_environment_and_first_build.ipynb`](01_environment_and_first_build.ipynb) creates or updates the `cofkit` Conda environment, registers its kernel, and builds a first TAPB–TFB imine COF.
2. [`02_topologies_connectivities_and_linkages.ipynb`](02_topologies_connectivities_and_linkages.ipynb) compares a compact set of 2D/3D topologies, connection combinations, and linkage chemistries.
3. [`03_zeopp_pore_analysis.ipynb`](03_zeopp_pore_analysis.ipynb) discovers Notebook 1's CIF and runs cofkit's Zeo++ pore-analysis wrapper.

Executable cells are CLI-first shell blocks. The corresponding Python API examples are present as fully commented code cells.
