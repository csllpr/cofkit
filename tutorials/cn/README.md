# Jupyter 教程

语言：[English](../en/README.md) | **中文**

平台：**Bash（Linux/macOS/WSL）** | [原生 Windows](../windows/cn/README.md)

请从仓库根目录启动 Jupyter，并按以下顺序学习：

1. [`01_environment_setup.ipynb`](01_environment_setup.ipynb) 创建或更新 `cofkit` Conda 环境，并注册其 Jupyter 内核。
2. [`02_first_cof_build.ipynb`](02_first_cof_build.ipynb) 首次构建 TAPB–TFB 亚胺 COF。
3. [`03_topologies_connectivities_and_linkages.ipynb`](03_topologies_connectivities_and_linkages.ipynb) 用一组精简示例比较二维/三维拓扑、连接数组合和连接键化学。
4. [`04_zeopp_pore_analysis.ipynb`](04_zeopp_pore_analysis.ipynb) 自动查找 Notebook 2 生成的 CIF，并运行 cofkit 的 Zeo++ 孔隙分析封装。

可执行单元格采用 CLI 优先的 Shell 命令。相应的 Python API 示例均放在完全注释掉的代码单元格中。
