# Jupyter 教程

语言：[English](../en/README.md) | **中文**

请从仓库根目录启动 Jupyter，并按以下顺序学习：

1. [`01_environment_and_first_build.ipynb`](01_environment_and_first_build.ipynb) 创建或更新 `cofkit` Conda 环境、注册其 Jupyter 内核，并首次构建 TAPB–TFB 亚胺 COF。
2. [`02_topologies_connectivities_and_linkages.ipynb`](02_topologies_connectivities_and_linkages.ipynb) 用一组精简示例比较二维/三维拓扑、连接数组合和连接键化学。
3. [`03_zeopp_pore_analysis.ipynb`](03_zeopp_pore_analysis.ipynb) 自动查找 Notebook 1 生成的 CIF，并运行 cofkit 的 Zeo++ 孔隙分析封装。

可执行单元格采用 CLI 优先的 Shell 命令。相应的 Python API 示例均放在完全注释掉的代码单元格中。
